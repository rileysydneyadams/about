# AboutMe - Digital Bio Page

This repository hosts the personal digital bio page for Riley Sydney Adams. It is a single-page lifestyle portfolio designed to showcase creative work, travel history, and personal branding, intended to be linked in social media profiles like Instagram.

## Features
- **Professional Bio:** Highlights my role as a Digital Content Creator, Lifestyle Influencer, and Freelance Creative Consultant.
- **Interactive Sections:** Includes sections for my story, career journey, lifestyle, travel history, and personality.
- **Responsive Design:** Built with a modern aesthetic featuring California minimalism, optimized for both desktop and mobile viewing.
- **Social Integration:** Direct links to Instagram, Facebook, and professional email for networking.

## Tech Stack
- **HTML5:** Structuring the content.
- **CSS3:** Styling, animations (drifting orbs), and responsive layout.
- **JavaScript:** Handles scroll progress tracking and scroll-reveal animations for a smooth user experience.

## Deployment
